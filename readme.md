# 🛒 Supermarket Sales Dashboard

An interactive data dashboard built with **Streamlit** that analyzes supermarket sales, customer behavior, product performance, and profitability.

## ✨ Features
- **KPI Cards** – Total sales, gross income, average rating, transactions, basket size.
- **Sales Overview** – By day, city, branch, payment method.
- **Product Analysis** – Top products by revenue/quantity, heatmap of sales by day.
- **Customer Insights** – Member vs normal, gender, average basket size.
- **Time Analysis** – Hourly, monthly, weekday vs weekend, time of day (morning/afternoon/evening/night).
- **Profit & Ratings** – Gross income per product, rating distribution, sales vs rating scatter plot.
